# C Project – Command-Line Calculator

This project is a simple command-line calculator written in C that demonstrates core programming concepts such as control flow, input validation, and arithmetic operations.

## Overview
The program allows users to perform repeated calculations using a menu-driven interface. It continues running until the user chooses to exit.

## Features
- Supports addition, subtraction, multiplication, and division
- Continuous execution loop until user exits
- Input validation for invalid operations
- Division-by-zero protection
- Formatted output for calculations

## Skills Demonstrated
- C programming fundamentals
- Control flow (loops and conditionals)
- User input handling
- Defensive programming (error handling)
- Procedural design
