# Java Project 1 – Stack Operations & Generics

This project demonstrates the implementation and manipulation of stacks using both Java’s built-in Stack class and a custom generic stack.

## Overview
The program performs multiple stack-based operations on integers and strings, including filtering, sorting, and rearranging elements.

## Features
- Removes odd values from an integer stack
- Custom generic stack implementation using ArrayList
- Sorting stacks using a custom algorithm (no built-in sort)
- Moving a specific value to the bottom of the stack
- File input for dynamic data loading (Numbers.txt, Strings.txt)
- Works with both integers and strings using generics

## Key Concepts
- Stack data structure (LIFO)
- Generics (`<E>`) for reusable data structures
- Custom sorting logic using stack operations
- File I/O using Scanner
- Algorithm design without relying on built-in utilities

## Skills Demonstrated
- Data structures (Stacks)
- Generic programming
- Algorithmic problem solving
- File handling in Java
- Debugging and logical flow control

# Java Project 2 – Pinball Machine Simulation

This project simulates a pinball machine using object-oriented programming and file-driven input.

## Overview
The program creates a 2D grid representing a pinball machine. Targets are placed on the grid, and gameplay is simulated by reading moves from a file. The program tracks hits and calculates a running score.

## Features
- 2D grid-based playing field
- Target objects with type, ID, points, and hit tracking
- File input for:
  - Target placement
  - Gameplay simulation
- Real-time score tracking
- Sorting targets by number of hits
- Formatted output of results

## Key Classes
- **PinballMachine**
  - Manages the grid and gameplay
- **Target**
  - Stores target data and tracks hits
  - Implements `Comparable` for sorting
- **Main Logic**
  - Reads input files and drives simulation

## Skills Demonstrated
- Object-oriented programming (classes, encapsulation)
- 2D arrays and data modeling
- File input/output
- Sorting using `Comparable`
- Algorithmic thinking and simulation design
