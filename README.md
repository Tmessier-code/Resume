# Technical Projects Portfolio

This repository contains programming projects demonstrating core concepts in data structures, object-oriented programming, file processing, and low-level procedural logic.

## Projects

### C Project – Calculator
A command-line calculator written in C that performs basic arithmetic operations with input validation and continuous execution.

### Java Project 1 – Stack & Generics
A Java program that implements stack operations, including filtering, sorting, and manipulating both integer and string data using a custom generic stack.

### Java Project 2 – Pinball Machine Simulation
A Java application that simulates a pinball machine using a 2D grid of targets, file-driven input, and object-oriented design.

## Skills Demonstrated
- Data structures (Stacks, Arrays, ArrayLists)
- Object-oriented programming (Java)
- Generics and type abstraction
- File input/output handling
- Algorithmic problem-solving
- Input validation and control flow
- Debugging and program design
